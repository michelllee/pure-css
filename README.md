## Install NodeJS

https://nodejs.org/en/download/

### NPM packages to install

- webpack
- webpack-cli
- webpack-dev-server
- css-loader
- html-loader
- html-webpack-plugin
- mini-css-extract-plugin
- pug-html-loader

```
npm init
npm install webpack
```

### Create a blank file

```
touch .gitignore
```

### Run Scripts

```
npm run watch
```
